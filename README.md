- Input interativo



# Como executar o projeto

## Front end web
Pré-requisitos: npm / yarn

```bash
# clonar repositório
git clone https://github.com/diwberg/react-pomodoro

# entrar na pasta do projeto front end web
cd react-pomodoro

# instalar dependências
npm install

# executar o projeto
npm run dev
```
